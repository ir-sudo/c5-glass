# C5 — helloc5.com

The website and live demo tools for **C5**, a done-for-you lead-recovery service for glass and window shops. Live at **[helloc5.com](https://helloc5.com)**.

C5 helps glass shops catch the jobs they're losing — missed calls, slow quotes, and leads that slip through the cracks — with simple tools that fit how they already work.

## What's here

The static site behind helloc5.com, plus a few standalone demos:

- **`/`** — the main C5 site
- **`build-cascade/`** — landing page for Cascade Window & Door (San Jose)
- **`build-mars/`** — landing page for North Bay Glass Company (Bay Area)
- **`quote/`** — Instant Quote Tool: a live demo that prices a job on the spot
- **`missed-call/`** — Missed-Call Text-Back: a live demo that auto-replies to missed calls

## Stack

Plain static site — HTML, CSS, and vanilla JavaScript, served via GitHub Pages on a custom domain. No build step.
